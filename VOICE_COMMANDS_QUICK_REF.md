# Voice Commands - Quick Reference

## How to Use

1. Open the app
2. Click the floating AI assistant button (bottom right)
3. Click the microphone button to start
4. Say a command or ask a question
5. The assistant will respond with audio and take action

## Tips for Better Recognition

1. **Speak Clearly:** Slow down and pronounce words clearly
2. **Quiet Environment:** Minimize background noise
3. **Microphone:** Position microphone close to mouth (6 inches)
4. **Simpler Commands:** Start with simple commands like "Go home"
5. **Natural Speech:** Say commands like you would normally speak

## Troubleshooting

### "Microphone not working"
- Check browser permissions (allow microphone access)
- Try refreshing the page
- Ensure microphone is connected
- Try different browser (Chrome recommended)

### "Command not recognized"
- Try simpler, shorter command
- Speak more clearly
- Check language is correct (dropdown in assistant)
- Try asking as a question instead

### "No audio response"
- Check volume is turned on
- Allow speaker/audio permissions
- Try text response instead
- Test with simpler command first

### "Form not filling"
- Speak one field at a time
- Say only the value (e.g., "5" not "five")
- For dates say "5 January 2024"
- For options, say exact option name

## Advanced Tips

### Chaining Commands
1. "Go to crops"
2. "Show my farms"
3. "Add crop" (all in sequence)

### Form Filling
Say values one by one:
- Field 1: "Wheat"
- Field 2: "5 acres"
- Field 3: "Drip irrigation"

### Getting Help
- Say "Help" to see available commands
- Say "Show suggestions" for page-specific commands
- Use manual buttons if voice doesn't work

### Feedback
- Green checkmark = Command understood
- Red X = Retry or use manual option
- Toast notification = Action confirmation
